# beanbins

Simple tool to generate parametric 3D-printable bins.

👉 https://beanbins.xyz

## What it does

- Adjust bin dimensions
- Preview in browser
- Download STL
- Print

No installs. No accounts.

## Notes

- Works best on desktop
- Mobile UI is limited
- STL generation may take a few seconds (cloud backend)

## Source

Frontend: https://github.com/queezz/bin-generator-frontend  
Backend: https://github.com/queezz/bin-generator-server
Generator: https://github.com/queezz/bin-generator

## Network note

Some managed networks (e.g. universities, corporate Wi-Fi) may block the site or show security warnings due to HTTPS inspection (certificate injection).

If you see “connection is not private” or the site is blocked:
- try a different network (mobile hotspot, home Wi-Fi)
- or use GitHub pages directly: https://github.com/queezz/bin
