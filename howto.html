🫘 Bin

How to use Beanbins

  1. Set dimensions — Enter X, Y, and H in mm.
  2. Click Generate — The 3D model will be created.
  3. Download STL and slice — Download the file and open it in your slicer.

First generation may take a few seconds if the server is starting up. After that, it's fast.

Model details

  • Wall thickness — Fixed at 1.2 mm. Works well for both 0.4 mm and 0.6 mm nozzles.
  • Design focus — Optimized for fast printing with 0.6 mm nozzles. Tested on Voron and Bambu A1 mini.
  • Print behavior —
    • Seam artifacts may appear if tuning is off
    • Vase mode: very clean results
    • Well-tuned printers: reliable in both modes
  • Vase mode recommendation — 0.6 mm nozzle, 0.3 mm layer height, 1.0–1.2 mm line width.
  • Size limits —
    Minimum: 15 mm
    Maximum: 300 mm
  • Connectors ("ears") — Work in progress, optimizing for the vase mode. Geometry will change. Future versions will support bin-to-bin connections.
  • Future improvements —
    • Improved wall structure (stronger than plain walls)
    • Better vase-mode compatibility
    • Optional grid system
  • Goal — Quickly generate simple, printable bins to exact size.
