How to use Beanbins

  1. Set dimensions — Enter X, Y, and H in mm.
  2. Click Generate — The 3D model will be created.
  3. Download STL and slice — Download the file and open it in your slicer.

Main site: https://beanbins.xyz
Fallback (GitHub Pages): https://queezz.github.io/bin/

Some managed networks (e.g. universities) may intercept HTTPS traffic and block custom domains. This can trigger browser warnings even for safe sites. The GitHub Pages version is usually not affected and can be used as a fallback.
